    🔧
    Custom Headers

    Add custom headers using the prefix h_:

    ?url=STREAM_URL&h_user-agent=CustomUA&h_referer=https://site.com&h_authorization=Bearer token123

    Supported headers:

    • • h_user-agent - Custom User Agent
    • • h_referer - Referring site
    • • h_authorization - Authorization token
    • • h_origin - Origin domain
    • • h_* - Any custom header
